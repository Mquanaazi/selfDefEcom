path method description
USERS TABLE
/users/login POST allows the user to sign in
/users/ GET Gets all of the users
/users/userID GET Gets a single user by userID
/users/ POST Adds a user
/users/userID DELETE Deletes a single user by userID
/users/userID PATCH edits all info of the user by userID
products TABLE
/products/ GET Gets all of the product
/products/productID GET Gets a single product by ID
/products/ POST Adds a product
/products/productID DELETE Deletes a single product by productID
/products/productID PATCH edits all info of the product by productID
