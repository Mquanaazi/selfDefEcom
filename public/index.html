Method Path Description
USERS TABLE
POST /users/login allows the user to sign in
GET /users/ Gets all of the users
GET /users/userID Gets a single user by userID
POST /users/signUp Adds a user to the database
POST /users/login logs a user
DELETE /users/userID Deletes a single user by userID
PATCH /users/userID edits all info of the user by userID
PRODUCTS TABLE
GET /products/ Gets all of the product
GET /products/productID Gets a single product by ID
POST /products/ Adds a product in to the database
DELETE /products/productID Deletes a single product by productID
PATCH /products/productID edits all info of the product by productID
